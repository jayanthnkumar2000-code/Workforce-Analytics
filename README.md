# Workforce-Analytics
📊 Workforce & Labour Cost Analytics

 Project Objective
This repository contains a comprehensive Power BI analytics solution designed to evaluate workforce efficiency, monitor departmental absenteeism, and optimize labor-to-revenue ratios. The dashboard provides executive leadership and HR managers with actionable insights to control labor costs and improve overall operational productivity.


Executive View: Highlights global absenteeism and baseline workforce efficiency.
Financial View: Breaks down Labor Cost Per Unit and Labor to Revenue ratios.
